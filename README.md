# Coq to OCaml to JS

## Prerequirements

- Make sure you have the required dependencies installed:
  - OCaml version 4.08.1
  - Coq version 8.10.0
  - opam version 2.0.5
  - Node.js version 12.13.0
  - yarn version 1.19.1

## Overview

```
Coq Code
  |
  | (Use Coq Compiler)
  v
OCaml Code
  |
  | (Use BuckleScript)
  v
Optimized Javascript Code
  |
  | (Use Rollup, Closure Compiler, Terser)
  v
More Optimized JavaScript Code
```
